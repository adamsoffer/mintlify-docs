Livepeer is the open GPU network for real-time AI video. It connects developers building AI-powered video applications with a global pool of independent GPU providers — at 60–85% lower cost than centralized cloud alternatives. This documentation covers everything you need to authenticate, make your first inference call, and build production applications on the network.

## What you can build

Livepeer supports real-time GPU inference on live video streams. Common use cases include:

- **AI-generated worlds** — Live environments rendered frame-by-frame from user inputs at 60 FPS
- **AI avatars** — Real-time video representations of AI agents for tutoring, telepresence, and conversational interfaces
- **Live video intelligence** — Object detection, scene classification, and analytics processed during the stream
- **Prompt-driven style transfer** — Transform a live stream's lighting, environment, or visual style via text prompt
- **Live transcoding** — Adaptive bitrate transcoding across a global GPU network with sub-second latency

## Get started in three steps

<Steps>
  <Step title="Create an account">
    Sign up at the [Developer Dashboard](https://livepeer.org/dashboard) to get access to the network.
  </Step>
  <Step title="Get your API key">
    Generate an API key from your dashboard settings. You'll use this to authenticate all requests. See [Authentication](/authentication) for details.
  </Step>
  <Step title="Make your first call">
    Install the Python SDK and run your first inference job. The [Quick Start](/quickstart) walks you through the full process in under five minutes.
  </Step>
</Steps>

<Note>
  The Livepeer network operates at an estimated 90–95% uptime. It is designed for builders who value cost and openness — not for workloads requiring enterprise SLAs. See [API Overview](/api-reference/overview) for details on reliability and pricing.
</Note>
